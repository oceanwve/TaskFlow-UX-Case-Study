# TaskFlow-UX-Case-Study

## Overview
TaskFlow is a productivity and task management application designed to help users organize tasks, manage priorities, and improve daily productivity through a clean and intuitive user experience.

This repository contains the complete UX case study, documenting the design process from user research to the final high-fidelity prototype.
